# PrepWise

An AI-powered Mock Interview Platform designed and built from the ground up to help users practice real-time conversational interviews.

## Features
- Real-time voice conversational AI
- Automated scoring and tailored feedback
- Dynamic interview generation based on tech stack, seniority, and role
- Next.js 15, TailwindCSS, Firebase Auth, and Vapi.ai Integration

## Installation

1. Clone the repository
2. Run `npm install`
3. Add your environment variables in `.env.local`
4. Start the application with `npm run dev`
